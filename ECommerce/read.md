# ECommerce Project - Feature Checklist

## 👤 User Management & Interaction
- [x] **Role-Based Access Control**: Implementation of different user roles (e.g., ADMIN, CUSTOMER).
- [ ] **Notification System**: Real-time or email notifications for orders and updates.
- [x] **Address Management**: Support for multiple shipping and billing addresses.
- [ ] **Product Reviews**: Allowing users to rate and review products.

## 📦 Product Catalog (Advanced)
- [x] **Category Hierarchy**: Category -> SubCategory structure.
- [x] **Product Management**: Products with multi-image support.
- [x] **Variant System**: Dynamic product variants (e.g., size, color).
- [x] **Inventory/Stock Management**: Tracking stock levels per variant.

## 🛍️ Cart & Checkout Workflow
- [x] **Shopping Cart**: Persistent cart for logged-in users with status-based filtering.
- [x] **Wishlist**: Persistent wishlist with the ability to move items to/from cart.
- [ ] **Order Management**: Order placement and status tracking.
- [x] **Payment Integration**: Secure payment gateway integration.
- [ ] **Refund System**: Handling returns and refunds.
- [ ] **Order History**: User's past purchases and transaction details.
